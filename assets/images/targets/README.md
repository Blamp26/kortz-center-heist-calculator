# Local target image library

Store only manually approved local images here.

Recommended layout:

- `primary/<target-id>.webp`
- `secondary/paintings/<target-id>.webp`
- `secondary/small-exhibits/<target-id>.webp`
- `secondary/coquard-eggs/<target-id>.webp`
- `secondary/reinforced-displays/<target-id>.webp`

Update `data/image-manifest.json` after adding each file.
